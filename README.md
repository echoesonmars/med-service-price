# MedServicePrice — Агрегатор цен на медицинские услуги

Архитектура проекта разделена на три автономные части:

## Структура проекта

*   **`/web`** — Фронтенд-приложение. Интерфейс для быстрого поиска и фильтрации услуг.
*   **`/backend`** — Основное API проекта. Отвечает за базу данных, кэш, бизнес-логику и быструю выдачу данных на клиент.
*   **`/agent`** — Отдельный изолированный сервис (слой оркестрации). Занимается асинхронным сбором данных, AI-нормализацией прайсов через LLM и маппингом услуг.

## Запуск проекта

*   Папка `/web` (Next.js): `npm run dev`
