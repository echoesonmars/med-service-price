# /chat — ИИ-Ассистент и чат-сервис

Для оптимизации архитектуры, простоты развертывания и экономии ресурсов (запуск без выделения дополнительного порта), API и логика ИИ-чатбота интегрированы непосредственно во фронтенд-слой.

*   **Оригинальный код API роута чата:** [`/web/app/api/chat`](../web/app/api/chat)
*   **Спецификация и стек:** TypeScript + Vercel AI SDK для стриминга ответов и генерации UI.
